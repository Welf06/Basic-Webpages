Guess the number

OBJECTIVE:

  • Computer will generate a random number from 1 to 10.
  • Player should guess the random number.


How to play:

  • Enter the random number when u load the website in the asked input.
  • To try again reload the website.



Results:

  • If guessed correctly then system responds with a success message.
  • If guessed incorrectly then system responds with the try again message.
